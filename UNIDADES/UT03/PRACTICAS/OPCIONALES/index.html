PRÁCTICAS UT03. (OPCIONALES)

-OPT0301 Cadenas (VI): Crea una función isHeterograma(str) que devuelva true si la cadena pasada como parámetro es un heterograma y false en caso contrario.

-OPT0304 Arrays (VII): Crea una función getUpperCase( arr ) que reciba como parámetro un arrays con strings y devuelve un array que únicamente contenga los strings cuyas letras sean todas mayúsculas.

-OPT0305 Arrays (VIII): Crea una función maxString( arr ) que tome como parámetro un array de strings y devuelva la cadena de mayor longitud.

-OPT0306 Arrays (IX): Crea una función getNames( arr ) que tome como entrada un array de cadenas con nombres (nombre y apellidos) y devuelva un array de objetos de la forma { nombre: 'xxx', apellidos: 'yyy zzz' }. Vamos a suponer que tanto los nombres como cada uno de los apellidos solo tienen una única palabra.

-OPT0307 Repaso (I): Crea una función reverseWords(str) que tome una cadena strcomo entrada y devuelva la misma cadena invirtiendo el orden de las letras de cada palabra.

-OPT0308 Repaso (II):Crea una función arrayDiff( a, b ) que tome como parámetros dos arrays y devuelva el resultado de eliminar del primero todos los elementos que se encuentren en el segundo.

-OPT0309 Repaso (III): Crea una función digitalRoot(a) que reciba como parámetro un número y devuelva la suma recursiva de todos sus dígitos. Es decir, sumará sus dígitos y, en caso de que el resultado tenga más de un dígito seguirá sumándolos hasta que el resultado sea un único dígito.

-OPT0310 Repaso (IV): Desarrolla una función findNextPower( n, pow) que, dado un número n, devuelva la potencia pow de otro número más próxima que sea superior a n.

-OPT0312Repaso (VI): Crea una función XO( str ) que tome como parámetro una cadena y devuelva true si la cadena tiene el mismo número de X que de O (sin distinguir entre mayúsculas y minúsculas). La cadena puede tener cualquier otro carácter que se ignorará.

-OPT0313 Repaso (VII): Crea una función cakes( recipe, ingredients ) donde recipe es un objeto con los ingredientes y cantidades que hacen falta para realizar una receta y ingredients es otro objeto con los ingredientes que hay disponibles. La función debe devolver un número que corresponderá con el número de recetas que se pueden realizar con los ingredientes disponibles.
